# ENS Talk 2019-11-10

### Getting Started

```
$ git clone https://github.com/swaldman/eth-command-line.git --branch 0.3.4
$ cd eth-command-line/
$ ./sbtw
```

If you already have `eth-command-line` installed on your machine, please upgrade

```
$ cd eth-command-line
$ git checkout 0.3.4
$ ./sbtw
```

You should be ready to go!
